File Types

Overview Raster Vector File Types


Now that we have discussed Vector and Raster graphics in detail, lets learn the different file types for each.

The most common Raster file types include JPG, GIF, PNG, TIF, BMP, and PSD.

The most common Vector file types are AI, CDR, and SVG.

Both Rasters and Vectors can be rendered in EPS and PDF format.

Vector file sizes usually remain small because the only information they need to contain are their mathematical formulas.

Raster file sizes can be larger because they must contain all the information necessary to render the image (pixels, colors, arrangement of pixels, etc.)
