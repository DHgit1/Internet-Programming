Raster vs. Vector Images and File Types

Overview Raster Vector File Types


If you are a Computer Scientist or Graphic Designer, you may have heard of Raster or Vector graphical images.

Wether you know it or not, these image types make up a large portion of what you view on the internet.

Before we go into depth of each image type, what exactly are Raster and Vector images?


Raster Graphics:

Raster images, also known as bitmaps, are made of individual pixels of color. Pixels are placed in a rectangular grid where each color pixel contributes to the overall image.

Vector Graphics:

Vector graphics are made up of paths, each with a mathematical formula (or Vector) which tells the path how it is shaped and what color it is bordered with or filled by.
